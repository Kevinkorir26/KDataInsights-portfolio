Projects

1. IPL Win Probability Predictor

Data Science Project at Sankyana Consultancy Services. This project involved building a machine learning model to predict the win probability of an IPL team in real time based on match conditions such as overs, wickets, and score. I used historical ball-by-ball data and match-level statistics, performed data wrangling, feature engineering, and implemented a classification model using logistic regression and random forests.

2. Fraud Detection System

Developed a fraud detection system for financial transactions. Using a large, imbalanced dataset, I performed EDA, handled class imbalance with SMOTE, and built a Random Forest model. I focused on model interpretability and deployed evaluation metrics like precision-recall and ROC-AUC.

3. Customer Churn Analysis

Created a predictive model to identify customers likely to leave a subscription-based service. This involved analyzing behavior data, identifying churn signals, and building a logistic regression model.

4. Sales Forecasting Dashboard

Built an interactive dashboard to forecast monthly sales using ARIMA and Exponential Smoothing, visualized using Plotly and Dash. Enabled stakeholder interaction by filtering regions and product lines.
